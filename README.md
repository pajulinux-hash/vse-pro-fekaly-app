# Historické stroje Fimas

Web prezentující sbírku historické zemědělské techniky firmy Fimas s.r.o. z Nového Veselí.

🌐 **Web:** [historicke-stroje.cz](https://www.historicke-stroje.cz)

## Struktura

- `index.html` — kompletní web (HTML + CSS + JS v jednom souboru)
- `PJF_5591male.webp` — fotka zakladatele
- `vercel.json` — konfigurace pro Vercel hosting

## Úprava obsahu

Obsah webu (stroje, galerie, výstavy) se spravuje přímo v souboru `index.html` v sekci označené:

```
JEDNODUCHÝ CMS - ZDE UPRAVUJTE OBSAH WEBU
```

### Přidání nového stroje
Najděte `var stroje = [` a přidejte nový blok.

### Přidání fotky do galerie
Najděte `var galerie = [` a přidejte řádek.

### Přidání nové výstavy
Najděte `var vystavy = [` a přidejte nový blok (nejnovější nahoře).
